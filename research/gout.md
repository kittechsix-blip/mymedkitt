## ACR/EULAR 2015 Gout Classification Criteria

**Source:** Neogi et al., Arthritis & Rheumatology 2015;67(10):2557-2568

### Step 1: Entry Criterion (REQUIRED)
At least 1 episode of peripheral joint or bursal swelling, pain, or tenderness

### Step 2: Sufficient Criterion (Diagnostic)
**Presence of MSU crystals in symptomatic joint/bursa or tophus = GOUT (no scoring needed)**

### Step 3: Scoring Criteria (if Step 2 not met)

**Threshold: Score of 8 or more = Gout classification**

#### Clinical Domain

| Category | Criteria | Points |
|----------|----------|--------|
| **Pattern of Joint Involvement** | | |
| | Joint(s) other than ankle, midfoot, or 1st MTP | 0 |
| | Ankle or midfoot (mono/oligoarticular, without 1st MTP) | +1 |
| | 1st MTP involvement (mono/oligoarticular) | +2 |
| **Characteristics of Episode** | | |
| | None of the below | 0 |
| | Erythema over affected joint | +1 |
| | Can't bear touch or pressure | +1 |
| | Great difficulty walking/using joint | +1 |
| | (Maximum 3 points for characteristics) | |
| **Time Course of Episode** | | |
| | No typical episodes | 0 |
| | 1 typical episode (peak <24h, resolution <14d) | +1 |
| | Recurrent typical episodes | +2 |
| **Tophus** | | |
| | Absent | 0 |
| | Present (draining or chalk-like nodule under skin) | +4 |

#### Laboratory Domain

| Category | Criteria | Points |
|----------|----------|--------|
| **Serum Urate** | | |
| | <4 mg/dL (<0.24 mmol/L) | -4 |
| | 4 to <6 mg/dL (0.24 to <0.36 mmol/L) | 0 |
| | 6 to <8 mg/dL (0.36 to <0.48 mmol/L) | +2 |
| | 8 to <10 mg/dL (0.48 to <0.60 mmol/L) | +3 |
| | >=10 mg/dL (>=0.60 mmol/L) | +4 |
| **Synovial Fluid Analysis** | | |
| | MSU negative by polarized microscopy | -2 |
| | Not done | 0 |

#### Imaging Domain

| Category | Criteria | Points |
|----------|----------|--------|
| **Evidence of Urate Deposition** | | |
| | Not done or absent | 0 |
| | Double contour sign on ultrasound OR urate on DECT | +4 |
| **Gout-Related Joint Damage** | | |
| | Not done or absent | 0 |
| | Radiographic erosion (cortical break, sclerotic margin, overhanging edge) | +4 |

**Performance:** Sensitivity 92%, Specificity 89%

## Acute Flare Treatment Algorithm

**Sources:** ACR 2020 Guidelines, EULAR 2016 Recommendations

### First-Line Agents (Choose based on contraindications)

```
ACUTE GOUT FLARE
       |
       v
Any contraindications to NSAIDs?
(CKD, GI bleed, CVD, anticoagulation)
       |
  +---------+---------+
  |                   |
  No                  Yes
  |                   |
  v                   v
NSAID            Contraindications to Colchicine?
                 (Severe CKD, P-gp/CYP3A4 inhibitors)
                      |
                 +----+----+
                 |         |
                 No        Yes
                 |         |
                 v         v
            Colchicine   STEROIDS
                         (oral, IA, or IM)
```

### Treatment Priority (ACR 2020 Strong Recommendations)

1. **NSAIDs** - First choice if no contraindications
2. **Colchicine** - Low-dose preferred; first choice if started within 36h of onset
3. **Glucocorticoids** - Oral, IA, or IM; first choice if NSAID/colchicine contraindicated
4. **IL-1 Inhibitors** - Reserved for refractory cases

### Key Principles

- Start treatment within 24 hours of symptom onset for best efficacy
- Topical ice is helpful as adjuvant treatment
- Can start ULT during acute flare (does not prolong symptoms if anti-inflammatory provided)
- Combination therapy acceptable for severe/polyarticular flares

## Drug Dosing with Renal Adjustments

### NSAIDs

#### Indomethacin
| Regimen | Dose |
|---------|------|
| Initial | 50 mg TID for 2-3 days |
| Taper | 25 mg TID until resolved |
| Duration | 5-7 days total |
| **Renal:** CrCl <60 mL/min = AVOID |

#### Naproxen
| Regimen | Dose |
|---------|------|
| Initial | 750 mg x1, then 250 mg q8h |
| Alternative | 500 mg BID for 3-5 days, then 250 mg BID |
| Duration | Until attack subsides |
| **Renal:** CrCl <60 mL/min = AVOID |

**NSAID Key Points:**
- All NSAIDs equally effective (no evidence indomethacin superior)
- Must use full anti-inflammatory doses
- Do not taper early
- Continue until attack fully resolved

### Colchicine

#### Acute Flare Treatment
| Renal Function | Dose |
|----------------|------|
| **Normal/Mild CKD** (CrCl >50) | 1.2 mg x1, then 0.6 mg 1 hour later (total 1.8 mg day 1) |
| **Moderate CKD** (CrCl 30-50) | Same dose, monitor closely |
| **Severe CKD** (CrCl <30) | Same dose, do NOT repeat course for 2 weeks |
| **Dialysis** | 0.6 mg x1 only, do NOT repeat for 2 weeks |

**Critical Timing:** Low-dose colchicine most effective if started within 36 hours of symptom onset

**Drug Interactions (AVOID):**
- Strong CYP3A4 inhibitors: clarithromycin, ketoconazole, ritonavir
- P-gp inhibitors: cyclosporine
- If on these drugs: reduce colchicine dose by 50% or AVOID

#### Prophylaxis Dosing (when starting ULT)
| Renal Function | Dose |
|----------------|------|
| Normal/Mild CKD | 0.6 mg daily or BID |
| Moderate CKD | 0.6 mg daily |
| Severe CKD | 0.3 mg daily |
| Dialysis | 0.3 mg twice weekly |

### Glucocorticoids

#### Oral Prednisone
| Regimen | Details |
|---------|---------|
| **Option 1 (Simple)** | 30-40 mg daily x 5 days, then stop |
| **Option 2 (Taper)** | 40-60 mg x 3 days, then taper 10-15 mg q3d over 10-14 days |
| **Alternative** | 0.5 mg/kg/day x 5-10 days, then taper over 7-10 days |

**Duration:** 5-14 days depending on severity

#### Intra-articular Steroids
| Joint | Triamcinolone Acetonide | Methylprednisolone |
|-------|------------------------|-------------------|
| Large (knee) | 40 mg | 40-80 mg |
| Medium (ankle, wrist) | 20-30 mg | 20-40 mg |
| Small (MTP, MCP) | 10 mg | 10-20 mg |

**Technique:** Aspirate joint first (therapeutic + diagnostic), then inject steroid

#### Intramuscular Steroids
- Triamcinolone 60 mg IM x1 (polyarticular or unable to take PO)
- Methylprednisolone 80-120 mg IM x1

### IL-1 Inhibitors (Second-Line/Refractory)

#### Anakinra (Kineret)
| Indication | Dose |
|------------|------|
| Acute flare (refractory) | 100 mg SC daily x 3-5 days |
| **Renal:** CrCl <30 = use every other day |

#### Canakinumab (Ilaris)
| Indication | Dose |
|------------|------|
| Acute flare (refractory) | 150 mg SC x 1 dose |
| Repeat | Not before 12 weeks |
| **Cost:** ~$11,900 per dose (vs ~$2 for triamcinolone) |

**IL-1 Inhibitor Indications:**
- NSAIDs, colchicine, AND steroids contraindicated or failed
- >=3 flares/year with inadequate response to standard therapy
- Infection risk must be considered

## Contraindication Screening

### NSAID Contraindications (AVOID)

| System | Contraindications |
|--------|-------------------|
| **Renal** | CKD with CrCl <60 mL/min, AKI |
| **GI** | Active peptic ulcer, GI bleed history, concurrent anticoagulation |
| **Cardiovascular** | Uncontrolled HTN, CHF, recent MI, CAD |
| **Other** | NSAID allergy, aspirin-exacerbated respiratory disease, 3rd trimester pregnancy |

### Colchicine Contraindications

| Category | Details |
|----------|---------|
| **Absolute** | Combined severe renal AND hepatic impairment |
| **Relative** | CrCl <30 (reduce dose/frequency) |
| **Drug Interactions** | Strong CYP3A4 inhibitors, P-gp inhibitors |
| **Caution** | Elderly, muscle disease, statin use (myopathy risk) |

### Steroid Contraindications (Relative)

| Category | Details |
|----------|---------|
| **Diabetes** | Will cause hyperglycemia; warn patient, adjust meds |
| **Active infection** | Relative contraindication; weigh risk/benefit |
| **Uncontrolled HTN** | May worsen |
| **Psychiatric history** | May cause mood changes, insomnia |
| **GI ulcer** | Less risk than NSAIDs, but consider PPI |

## Pseudogout (CPPD)

### Overview

Calcium pyrophosphate deposition disease (CPPD), also called pseudogout, is characterized by CPP crystal deposition in joint cartilage. Most common cause of inflammatory arthritis in patients >60 years.

### Clinical Phenotypes

| Phenotype | Features |
|-----------|----------|
| **Acute CPP Arthritis** ("Pseudogout") | Acute monoarticular attack resembling gout; knee most common |
| **Chronic CPP Arthritis** | Polyarticular, resembles RA ("pseudo-RA") |
| **Osteoarthritis with CPPD** | OA pattern with chondrocalcinosis on imaging |
| **Asymptomatic** | Incidental chondrocalcinosis on X-ray |

### Diagnosis

#### Crystal Identification (Gold Standard)
- **Shape:** Rhomboid or rod-shaped (shorter than MSU)
- **Birefringence:** Weakly POSITIVE (blue when parallel to compensator)
- **Comparison:** MSU = needle-shaped, strongly NEGATIVE birefringent

| Feature | Gout (MSU) | Pseudogout (CPP) |
|---------|------------|------------------|
| Shape | Needle | Rhomboid/rod |
| Birefringence | Strong negative | Weak positive |
| Parallel to compensator | Yellow | Blue |
| Perpendicular | Blue | Yellow |

### Imaging Findings

| Modality | Findings |
|----------|----------|
| **X-ray** | Chondrocalcinosis: punctate/linear calcifications in cartilage |
| **Classic Locations** | Knee meniscus, triangular fibrocartilage of wrist, pubic symphysis |
| **Ultrasound** | Hyperechoic deposits within cartilage (vs gout double contour on surface) |
| **Sensitivity** | X-ray detects only ~40% of clinically significant CPPD |

### Treatment Differences from Gout

| Aspect | Gout | CPPD |
|--------|------|------|
| **Acute treatment** | NSAIDs, colchicine, steroids | Same options |
| **Crystal dissolution** | ULT eliminates crystals | NO therapy dissolves CPP crystals |
| **Prophylaxis** | Colchicine 0.6 mg daily-BID effective | Colchicine 0.6 mg daily-BID may reduce flares |
| **Long-term** | Treat to serum urate target | Supportive care only |

### CPPD Treatment Protocol

**Acute Attack:**
1. Joint aspiration + IA corticosteroid (preferred if single joint)
2. NSAIDs (indomethacin, naproxen same doses as gout)
3. Colchicine (same dosing as gout)
4. Oral steroids if above contraindicated

**Chronic/Recurrent:**
- Colchicine 0.6 mg daily may reduce flare frequency
- Low-dose NSAIDs for chronic symptoms
- IL-1 inhibitors (anakinra) for refractory cases

### Secondary Causes to Screen For

| Condition | Test |
|-----------|------|
| Hyperparathyroidism | Calcium, PTH |
| Hemochromatosis | Ferritin, transferrin saturation |
| Hypomagnesemia | Magnesium |
| Hypothyroidism | TSH |
| Hypophosphatasia | Alkaline phosphatase |

## When to Aspirate vs Empiric Treatment

### Indications for Joint Aspiration (ASPIRATE)

| Scenario | Rationale |
|----------|-----------|
| **Suspected septic arthritis** | Cannot rule out on clinical grounds |
| **First presentation/uncertain diagnosis** | Need crystal confirmation |
| **Atypical joint involvement** | Not classic podagra |
| **Systemically ill patient** | Fever, immunocompromised |
| **Treatment failure** | Empiric therapy not working |
| **Monoarticular with effusion** | Therapeutic benefit + diagnostic |
| **Coexisting conditions** | RA, prior joint disease (higher septic risk) |

### Empiric Treatment Acceptable When

| Scenario | Rationale |
|----------|-----------|
| **Classic podagra** | 1st MTP, acute onset, self-limited history |
| **Known confirmed gout** | Prior crystal-proven attacks |
| **High clinical probability** | ACR/EULAR score >=8 |
| **Aspiration not feasible** | Outpatient, small joint |
| **No red flags for infection** | Afebrile, no immunocompromise |

### Clinical Decision Rule (Without Aspiration)

**7-Point Scoring Tool for Gout:**
- Score >=8: High probability (PPV 0.87) - empiric treatment reasonable
- Score 4-7: Intermediate - consider aspiration or rheumatology referral
- Score <=4: Low probability (NPV 0.95) - unlikely gout

## Synovial Fluid Analysis Interpretation

### WBC Count Classification

| Classification | WBC/mm3 | Interpretation |
|----------------|---------|----------------|
| **Non-inflammatory** | <200-2,000 | OA, trauma |
| **Inflammatory** | 2,000-50,000 | Crystal arthritis, RA, reactive |
| **Septic** | >50,000 | Bacterial infection (but overlap exists) |

### Crystal Arthritis Ranges

| Finding | Gout | CPPD |
|---------|------|------|
| Typical WBC | 10,000-70,000/uL | Similar |
| Range | 1,000-100,000+/uL | Similar |
| PMN % | Usually >50% | Usually >50% |

**Key Point:** WBC counts in gout can exceed 50,000/uL (up to 100,000+ reported). Cannot use WBC alone to rule out infection.

### Crystal Identification Summary

| Crystal | Shape | Birefringence | Color Parallel | Color Perpendicular |
|---------|-------|---------------|----------------|---------------------|
| **MSU (Gout)** | Needle | Strong negative | Yellow | Blue |
| **CPP (CPPD)** | Rhomboid | Weak positive | Blue | Yellow |

**Sensitivity of crystal identification:** 84% (if negative but clinical suspicion high, repeat on subsequent flare)

### Additional Synovial Fluid Tests

| Test | Utility |
|------|---------|
| **Gram stain** | Low sensitivity (~50%) for septic arthritis |
| **Culture** | Gold standard for septic arthritis |
| **Lactate** | >10 mmol/L strongly suggests infection; <4.3 mmol/L unlikely |
| **Glucose** | Low SF:serum ratio (<0.5) suggests infection |

## Gout vs Septic Arthritis Differentiation

### The Diagnostic Challenge

- Both present as acute monoarthritis
- WBC counts overlap significantly
- Crystals do NOT exclude infection (can coexist)
- Gout/CPPD joints are at INCREASED risk for septic arthritis

### Clinical Features Comparison

| Feature | More Likely Gout | More Likely Septic |
|---------|------------------|-------------------|
| **Age** | Any (peak 40-60) | Any |
| **Sex** | Male predominant | Either |
| **Joint** | 1st MTP classic | Knee most common |
| **Onset** | Hours (peak <24h) | Hours to days |
| **Fever** | Less common | More common (but absent in 50%) |
| **Serum urate** | Often elevated | Usually normal |
| **ESR/CRP** | Elevated | More elevated |
| **Blood WBC** | May be elevated | Often elevated |
| **History** | Prior gout attacks | IV drug use, skin infection, immunocompromise, joint surgery |

### Synovial Fluid Differentiation

| Parameter | Gout | Septic Arthritis |
|-----------|------|-----------------|
| **WBC** | 10,000-70,000 (can exceed 100k) | >50,000 typical (but can be lower) |
| **PMN %** | >50% typical | >90% typical |
| **Crystals** | MSU present | None (unless coexisting) |
| **Gram stain** | Negative | Positive ~50% |
| **Culture** | Negative | Positive (gold standard) |
| **Lactate** | Low/normal | >10 mmol/L |

### When You Cannot Distinguish

**High-Risk Features for Coexisting Infection:**
- Synovial WBC >85,000/mm3 (even with crystals present)
- Fever >38.5C
- Immunocompromise (DM, steroids, HIV)
- Recent joint surgery/injection
- Bacteremia/endocarditis
- Skin infection overlying joint

**Management When Uncertain:**
1. Send synovial fluid for Gram stain AND culture
2. Hold antibiotics until cultures obtained if clinically stable
3. If toxic-appearing: treat for BOTH (steroids + antibiotics pending culture)
4. Repeat aspiration in 24-48h if culture pending
5. Rheumatology and/or Orthopedic consultation

## Flare Prophylaxis When Starting ULT

### Why Prophylaxis is Needed

- Initiating ULT (allopurinol, febuxostat) mobilizes tissue urate
- Flares occur in up to 75% of patients in first 6 months WITHOUT prophylaxis
- With prophylaxis: significantly fewer flares (0.5 vs 2.9 flares over 6 months)

### ACR 2020 Recommendations

| Recommendation | Strength |
|----------------|----------|
| Use prophylaxis when starting ULT | Strong |
| Duration: at least 3-6 months | Strong |
| Extend beyond 6 months if flares continue or tophi present | Conditional |

### Prophylaxis Options

| Agent | Dose | Notes |
|-------|------|-------|
| **Colchicine (preferred)** | 0.5-0.6 mg daily or BID | Adjust for renal function |
| **Low-dose NSAID** | Naproxen 250 mg BID | If colchicine contraindicated |
| **Low-dose prednisone** | 5-10 mg daily | If NSAID and colchicine contraindicated |

### Duration Guidelines

| Scenario | Recommended Duration |
|----------|---------------------|
| **No tophi** | 3-6 months after reaching SUA target |
| **Resolved tophi** | 6 months after reaching SUA target |
| **Visible tophi still present** | Continue until tophi resolve |
| **Ongoing flares** | Continue until flare-free for 3-6 months |

### Patient Counseling Points

1. **Expect flares initially** - does not mean treatment is failing
2. **Continue ULT through flares** - stopping prolongs disease
3. **Take prophylaxis as prescribed** - reduces flare severity and frequency
4. **Flares will decrease over time** - as crystal burden is reduced
5. **Long-term goal** - achieve and maintain serum urate <6 mg/dL (or <5 mg/dL if severe)

## Calculator Concepts

### 1. ACR/EULAR Gout Classification Score

**Purpose:** Classify patients as having gout for study inclusion (also useful for clinical probability)

**Inputs:**
- Joint pattern (other/ankle-midfoot/1st MTP)
- Episode characteristics (erythema, pressure intolerance, functional impairment)
- Time course (typical episodes, recurrence)
- Tophus present?
- Serum urate level
- SF analysis done? MSU result?
- Imaging done? Findings?

**Output:**
- Total score
- Classification (>=8 = Gout)
- Sensitivity/specificity information

---

### 2. Acute Gout Treatment Selector

**Purpose:** Select appropriate first-line agent based on contraindications

**Inputs:**
- CrCl/eGFR
- Active GI bleed or ulcer?
- CVD history (CHF, recent MI, uncontrolled HTN)?
- On anticoagulation?
- Diabetes?
- Active infection?
- Taking CYP3A4/P-gp inhibitors?
- Single joint with effusion?
- Time since symptom onset?

**Output:**
- Recommended agent(s) with dosing
- Agents to AVOID with reasons
- Consideration for IA injection if applicable

---

### 3. Colchicine Dosing Calculator (CKD Adjustment)

**Purpose:** Calculate appropriate colchicine dose based on renal function and indication

**Inputs:**
- Indication (acute flare vs prophylaxis)
- eGFR or CrCl
- Dialysis status
- Concurrent medications (check CYP3A4/P-gp interactions)

**Output:**
- Recommended dose
- Frequency
- Duration restrictions (e.g., do not repeat for 2 weeks)
- Drug interaction warnings

---

### 4. Synovial Fluid Interpreter

**Purpose:** Interpret synovial fluid analysis results

**Inputs:**
- WBC count (/mm3 or /uL)
- PMN percentage
- Crystal presence and type
- Gram stain result
- Culture result (if available)
- Glucose (and serum glucose for ratio)
- Lactate (if available)

**Output:**
- Classification (non-inflammatory/inflammatory/likely septic)
- Crystal diagnosis if applicable
- Risk assessment for concurrent infection
- Recommended next steps

---

### 5. Gout vs Septic Arthritis Probability

**Purpose:** Risk stratify monoarthritis for likelihood of septic vs crystal arthritis

**Inputs:**
- Age
- Temperature
- Joint involved
- Prior history of gout
- Immunocompromise status
- Recent skin infection or bacteremia
- IV drug use
- Recent joint surgery/injection
- Serum urate
- Blood WBC
- Synovial WBC (if available)
- Crystals present?

**Output:**
- Probability assessment (low/moderate/high risk for septic)
- Recommendation (empiric gout treatment OK vs aspiration mandatory)
- Red flags present
- Suggested workup
